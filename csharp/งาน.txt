# สวัสดีครับ ผมชื่อ นายอานัฐ ทองปน อายุ 28 ปี จบการศึกษาระดับปริญญาตรี จากมหาวิทยาลัยเทคโนโลยีราชมงคล อิสาน นครราชสีมา สาขาวิศวกรรมคอมพิวเตอร์
ปัจจุบันทำงานอยู่ที่บริษัท ไซแมทเทคโนโลยี จำกัด (มหาชน) ตำแหน่ง Senior .Net Web Programmer ครับ
ประสบการณ์การทำงานประมาณ 5 ปี ครับ เกี่ยวกับสายงาน .Net Web Programmer 
งานหลักที่ออกแบบและพัฒนาที่ผ่านมา จะเป็นเว็ปแอพพลิเคชั่น เป็นหลัก และอาจจะมี Window Service , Web Service บ้างตามความเหมาะสมของแต่ละส่วนงานที่ต้องใช้
เทคโนโลยีและภาษาที่ใช้ในการพัฒนาแอพลิเคชัน คือ ภาษา C# และ ASP.Net MVC ,  Entity Framework กับฐานช้อมูล SQL Server  และรายงานใช้ Crytals Report

โปรเจคหลัก คือ เว็ปแอพพลิเคชั่นที่พัฒนาขึ้นเพื่อสนับสนุนการดำเนินงานและบริงานงานของธุรกิจอินเตอร์เน็ต

โดยผู้ใช้งานจะเป็นคนนำเข้าข้อมูลสู่ส่วนกลางที่ตั้งอยู่ กทม. ผ่านทางเว็ปแอพพลิชั่นที่ได้พัฒนาขึ้น อย่างเช่น การสร้างข้อมูลลูกค้า ที่มาสมัครใช้งานอินเตอร์เน็ต การรับชำระเงิน ค่าอินเตอร์เน็ต
